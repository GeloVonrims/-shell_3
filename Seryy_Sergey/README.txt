# Конвертация видео в изображения

Данный скрипт позволяет конвертировать видеофайлы в отдельные изображения с помощью библиотеки OpenCV.

## Зависимости

- Python 3.x
- OpenCV (cv2)
- NumPy

## Использование

1. Разместите видеофайлы, которые вы хотите конвертировать, в одной директории с данным скриптом.

2. Запустите скрипт `movie_to_img.py`.

3. Следуйте инструкциям, введя путь к директории с видеофайлами.

4. Скрипт будет обрабатывать каждый видеофайл, найденный в указанной директории, и сохранять каждый кадр в виде отдельного изображения.

5. Изображения будут сохранены в директории, названной по имени видеофайла, с добавлением суффикса "-opencv".

## Дополнительные настройки

- `SAVING_FRAMES_PER_SECOND`: количество кадров, которые нужно сохранить в секунду (по умолчанию 5).
- `BLUR_THRESHOLD`: пороговое значение оценки размытия для сохранения кадра (по умолчанию 1000.0).

## Примечания

- Если в директории с видеофайлами уже есть папки с такими же именами, скрипт создаст папку с добавлением суффикса "-opencv" для избежания конфликтов и перезаписи файлов.

- Скрипт проверяет коэффициент размытия каждого кадра и сохраняет только те кадры, у которых коэффициент размытия удовлетворяет порогу `BLUR_THRESHOLD`.

- Имя каждого сохраненного изображения будет состоять из префикса "ssv_Battle_in_market_" и времени создания скрипта, а также продолжительности кадра в формате "чч-мм-сс-мс".

- Для конвертации видеофайла в изображения используется библиотека OpenCV (cv2) и ее функции для чтения и обработки видео.

- Возможно, вам потребуется настроить параметры `SAVING_FRAMES_PER_SECOND` и `BLUR_THRESHOLD` в соответствии с вашими потребностями.
